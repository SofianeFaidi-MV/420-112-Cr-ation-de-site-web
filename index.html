LES CHAÎNES DE CARACTÈRES

Cette page montre le format à utiliser pour ton site : le contenu du cours est directement intégré dans le HTML, avec un titre clair, des explications, des exemples de code et des exercices visibles sans ouvrir de PDF.

Chapitre — JavaScript

En JavaScript, une chaîne de caractères représente un texte. Elle peut contenir des lettres, des chiffres, des espaces et des symboles. Les chaînes sont utilisées pour afficher des messages, construire du contenu HTML, manipuler des formulaires et produire des réponses visibles à l’écran.

Les strings, c’est quoi ?

Une chaîne de caractères est une valeur textuelle. En JavaScript, on peut l’écrire avec des guillemets doubles, des guillemets simples ou des accents graves. Une chaîne peut être stockée dans une variable puis affichée, modifiée ou combinée avec d’autres valeurs.

Exemple :

let message = "Bonjour !";

Utilisation simple

Créer une chaîne et l’afficher dans la console ou dans la page.

let nom = "Alice"; console.log(nom);

Créer une phrase plus longue :

let phrase = "J’apprends JavaScript dans mon cours de création de sites web."; console.log(phrase);

Méthodes utiles de String

Une chaîne possède plusieurs méthodes pratiques pour lire ou transformer du texte.

length

Compter les caractères

let texte = "JavaScript"; console.log(texte.length);
toUpperCase()

Mettre en majuscules

let texte = "bonjour"; console.log(texte.toUpperCase());
toLowerCase()

Mettre en minuscules

let texte = "BONJOUR"; console.log(texte.toLowerCase());
trim()

Enlever les espaces inutiles

let texte = " salut "; console.log(texte.trim());

Concaténation

La concaténation consiste à assembler plusieurs morceaux de texte pour former une seule phrase. On peut utiliser l’opérateur +.

let prenom = "Sofia"; let ville = "Montréal"; let resultat = "Bonjour, je m’appelle " + prenom + " et j’habite à " + ville + "."; console.log(resultat);
Résultat attendu : Bonjour, je m’appelle Sofia et j’habite à Montréal.

Template literals

Une autre façon moderne d’insérer des variables dans une phrase consiste à utiliser les accents graves et la syntaxe ${variable}.

let produit = "Clavier"; let prix = 24.99; let message = `Le produit ${produit} coûte ${prix}$`; console.log(message);

Exercices

🧪 Exo 1 — Déclarer une chaîne

Déclare une variable appelée cours contenant le texte Création de sites web, puis affiche-la.
let cours = "Création de sites web"; console.log(cours);

🧪 Exo 2 — Longueur d’une chaîne

Crée une variable mot contenant JavaScript et affiche sa longueur avec length.
let mot = "JavaScript"; console.log(mot.length);
Résultat attendu : 10

🧪 Exo 3 — Mettre en majuscules

Transforme le texte bonjour les étudiants en majuscules.
let texte = "bonjour les étudiants"; console.log(texte.toUpperCase());

🧪 Exo 4 — Concaténation simple

Déclare les variables suivantes et affiche une phrase complète.
let nomProduit = "Clavier"; let quantite = 5; let prixUnitaire = 24.99; let message = quantite + " " + nomProduit + " à " + prixUnitaire + "$ l’unité"; console.log(message);
Résultat attendu : 5 Clavier à 24.99$ l’unité

🧪 Exo 5 — Template literal

Réécris l’exercice précédent avec un template literal.
let message = `${quantite} ${nomProduit} à ${prixUnitaire}$ l’unité`; console.log(message);
