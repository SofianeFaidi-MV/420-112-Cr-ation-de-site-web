Introduction à la programmation Web

Ce site regroupe les concepts essentiels du cours : structure HTML, mise en forme CSS, interactivité avec JavaScript, formulaires, DOM, tableaux, liens, responsive design et exercices pratiques. L’objectif est de fournir une page unique, lisible et navigable comme une vraie documentation de cours.

HTML

Construire la structure sémantique d’une page, organiser le texte, créer des liens, des tableaux, des formulaires et intégrer du multimédia.

CSS

Mettre en forme la page avec couleurs, polices, espacements, bordures, arrière-plans et techniques de mise en page réactive.

JavaScript

Ajouter de l’interactivité, manipuler le DOM, utiliser les variables, conditions, boucles, fonctions et objets pour rendre le site dynamique.

Introduction au Web

Vue d’ensembleCours

En développement Web, HTML sert à structurer l’information, CSS à habiller cette structure et JavaScript à ajouter de l’interactivité. Une page moderne ne se limite plus à afficher de l’information : elle réagit, valide, anime et met à jour son contenu selon les actions de l’utilisateur.

Rôle des trois technologies

  • HTML : le squelette de la page.
  • CSS : l’apparence visuelle et la mise en page.
  • JavaScript : la logique, l’interaction et la manipulation du contenu.
À retenirUne page bien construite sépare la structure, la présentation et le comportement.

Exemple d’enchaînement

Un utilisateur remplit un formulaire HTML, CSS l’affiche proprement et JavaScript vérifie les champs avant l’envoi.

Structure de base d’un fichier HTML

HTMLBalises

Le squelette minimal

Un document HTML contient généralement la déclaration du type de document, l’élément racine <html>, une section <head> pour les métadonnées et une section <body> pour le contenu visible.

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ma première page</title>
</head>
<body>
  <h1>Bonjour !</h1>
  <p>Bienvenue dans le cours.</p>
</body>
</html>

Éléments importants

  • <title> définit le titre de l’onglet du navigateur.
  • <meta charset> permet d’afficher correctement les caractères accentués.
  • <meta name="viewport"> est important pour les écrans mobiles.
Exercice Crée une page avec un titre principal, deux paragraphes et une ligne horizontale entre les deux.

Structuration du texte

TitresParagraphesMise en valeur

Organiser le contenu textuel

On utilise les titres <h1> à <h6>, les paragraphes <p>, les retours de ligne <br> et des balises comme <strong> ou <em> pour mettre en valeur des mots ou des idées.

<h1>Cours de création Web</h1>
<h2>Chapitre 1</h2>
<p>Le HTML sert à structurer une page Web.</p>
<p>On peut mettre un mot en <strong>évidence</strong> ou en <em>emphase</em>.</p>
Exemple Une bonne hiérarchie de titres aide l’utilisateur à lire la page et améliore aussi l’accessibilité.

Les liens hypertextes

NavigationAncre

Lier les pages et les sections

Le lien hypertexte est au cœur du Web. Il permet de relier des pages entre elles, d’ouvrir des ressources externes, d’accéder à une section d’une page ou même d’envoyer un courriel.

<a href="https://www.example.com" target="_blank">Visiter Example</a>

<a href="#section1">Aller à la section 1</a>

<h2 id="section1">Section 1</h2>
<p>Contenu de la section 1.</p>

Cas fréquents

  • Lien externe : href="https://..."
  • Lien interne : href="#id"
  • Courriel : href="mailto:adresse@site.com"

Les tableaux

Tabletheadtbody

Présenter l’information en rangées et colonnes

Les tableaux servent à organiser des données de manière lisible. Ils sont composés de rangées, de colonnes et de cellules. On distingue souvent l’en-tête du corps du tableau.

<table>
  <thead>
    <tr>
      <th>Film</th>
      <th>Année</th>
      <th>Genre</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Camping sauvage</td>
      <td>2004</td>
      <td>Comédie</td>
    </tr>
  </tbody>
</table>

Nos premiers pas avec CSS

SélecteursPropriétés

Pourquoi CSS ?

HTML crée le squelette de la page, mais CSS lui donne son apparence. On peut modifier les couleurs, les marges, la taille des éléments, les bordures, les polices et la disposition générale.

Où écrire le CSS ?

  • Inline : dans l’attribut style d’un élément.
  • Interne : dans une balise <style> dans le <head>.
  • Externe : dans un fichier .css lié avec <link>.
p {
  color: white;
  background-color: #203040;
  padding: 12px;
  border: 1px solid #4da3ff;
}
Bonne pratiqueLa méthode recommandée est le fichier CSS externe, plus facile à maintenir.

Polices de caractères et styles

font-familyfont-size

Choisir la bonne police

Le choix d’une police influence la lisibilité et l’identité visuelle d’un site. On utilise souvent une liste de polices de repli pour s’adapter aux systèmes de l’utilisateur.

body {
  font-family: Arial, Helvetica, sans-serif;
  font-size: 16px;
  line-height: 1.6;
}

h1 {
  font-weight: bold;
  font-style: normal;
}

Propriétés courantes

  • font-family
  • font-size
  • font-weight
  • font-style
  • text-align

Les couleurs et les images de fond

colorbackground-image

Utiliser la couleur avec cohérence

Les couleurs améliorent l’apparence d’une page lorsqu’elles sont utilisées avec intention. Elles doivent rester lisibles, harmonieuses et adaptées au public visé.

body {
  color: #f5f7ff;
  background-color: #0f1115;
  background-image: url('fond.jpg');
  background-size: cover;
  background-repeat: no-repeat;
}
ConseilPrivilégie un contraste suffisant entre le texte et l’arrière-plan pour faciliter la lecture.

Audio et vidéo HTML5

audiovideo

Intégrer du multimédia sans plugin

HTML5 introduit les éléments <audio> et <video>, qui permettent de lire des médias directement dans le navigateur.

<audio controls>
  <source src="musique.mp3" type="audio/mpeg">
  Votre navigateur ne supporte pas l'audio.
</audio>

<video controls width="480">
  <source src="demo.mp4" type="video/mp4">
  Votre navigateur ne supporte pas la vidéo.
</video>
À surveillerTous les navigateurs ne gèrent pas exactement les mêmes formats. Il est souvent utile de prévoir plusieurs sources.

Les formulaires

forminputbutton

Interaction utilisateur

Les formulaires servent à recueillir de l’information : texte, choix, courriels, messages, options uniques ou multiples. Le formulaire est défini par l’élément <form>.

<form action="traitement.php" method="post">
  <label for="nom">Nom</label>
  <input id="nom" name="nom" type="text" required>

  <label for="email">Courriel</label>
  <input id="email" name="email" type="email">

  <label for="message">Message</label>
  <textarea id="message" name="message"></textarea>

  <button type="submit">Envoyer</button>
</form>

Attributs à connaître

  • action : destination des données
  • method : get ou post
  • name : identifie le champ
  • required : champ obligatoire

Réactivité

ResponsiveViewport

Concevoir une page qui s’adapte

Un site réactif s’adapte aux téléphones, tablettes et écrans d’ordinateur. La balise viewport aide le navigateur à dimensionner correctement la page sur mobile.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
.cartes {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}

.carte {
  flex: 1 1 280px;
}
ExerciceCrée trois cartes alignées sur grand écran et empilées sur téléphone.

Introduction à JavaScript

ScriptInteractivité

Pourquoi JavaScript ?

JavaScript est un langage de programmation qui permet de rendre une page Web interactive. Il peut afficher des messages, réagir à des événements, manipuler des formulaires et modifier le contenu HTML via le DOM.

<script>
  alert("Bonjour et bienvenue !");
</script>

Quelques usages

  • Validation de formulaires
  • Modification dynamique du texte
  • Animations et interactions
  • Traitement des entrées utilisateur

Variables et constantes

letconstprompt()

Stocker des valeurs

Les variables servent à conserver des informations en mémoire. En JavaScript moderne, on utilise surtout let pour une valeur modifiable et const pour une valeur qui ne doit pas être réassignée.

let nom = "Sofiane";
const cours = "420-112";

alert("Bonjour " + nom);

let age = prompt("Quel âge as-tu ?");
console.log(age);

Règles de nommage

  • Pas d’espace dans le nom.
  • Éviter de commencer par un chiffre.
  • Respecter la casse : nom et Nom sont différents.

Méthodes sur les chaînes, fonctions mathématiques et dates

stringMathDate

Manipuler les chaînes

let texte = "Développement Web";
console.log(texte.length);
console.log(texte.toUpperCase());
console.log(texte.includes("Web"));
console.log(texte.substring(0, 13));

Fonctions mathématiques

console.log(Math.round(4.6));
console.log(Math.floor(4.6));
console.log(Math.ceil(4.1));
console.log(Math.random());

Dates

let aujourdHui = new Date();
console.log(aujourdHui.getFullYear());
console.log(aujourdHui.getMonth() + 1);
console.log(aujourdHui.getDate());

Structures de contrôle conditionnelles

ifelseswitch

Prendre des décisions

Une structure conditionnelle exécute un bloc de code selon une condition logique.

let note = 78;

if (note >= 60) {
  console.log("Réussi");
} else {
  console.log("Échec");
}
let jour = 3;

switch (jour) {
  case 1:
    console.log("Lundi");
    break;
  case 2:
    console.log("Mardi");
    break;
  case 3:
    console.log("Mercredi");
    break;
  default:
    console.log("Autre jour");
}
ExerciceÉcris un script qui affiche “majeur” si l’âge entré par l’utilisateur est supérieur ou égal à 18, sinon “mineur”.

Structures de répétition

forwhilefor...of

Répéter des instructions

for (let i = 1; i <= 5; i++) {
  console.log("Bonjour " + i);
}
let compteur = 0;
while (compteur < 3) {
  console.log("Compteur : " + compteur);
  compteur++;
}
let couleurs = ["rouge", "vert", "bleu"];
for (let couleur of couleurs) {
  console.log(couleur);
}
À retenirLa boucle for convient bien quand on connaît le nombre de répétitions à l’avance.

Le modèle DOM, les fonctions et les objets

DOMfunctionobject

Comprendre le DOM

Le DOM représente la page comme un ensemble de nœuds et d’objets manipulables. Chaque élément HTML peut être ciblé et modifié par JavaScript.

Fonctions

function saluer(nom) {
  return "Bonjour " + nom;
}

console.log(saluer("Marie"));

Objets

let etudiant = {
  nom: "Ali",
  programme: "Informatique",
  session: 1
};

console.log(etudiant.nom);

JavaScript et le DOM

getElementByIdevent

Modifier une page avec JavaScript

Grâce au DOM, JavaScript peut accéder au contenu, au style et aux attributs des éléments HTML, puis les modifier selon les actions de l’utilisateur.

<h1 id="titre">Titre initial</h1>
<button onclick="changerTitre()">Changer le titre</button>

<script>
function changerTitre() {
  document.getElementById("titre").textContent = "Titre modifié";
}
</script>
<input id="nom" type="text" placeholder="Votre nom">
<button id="btn">Afficher</button>
<p id="resultat"></p>

<script>
document.getElementById("btn").addEventListener("click", function () {
  const valeur = document.getElementById("nom").value;
  document.getElementById("resultat").textContent = "Bonjour " + valeur;
});
</script>
