Cours 420-112

Apprendre à créer des sites web modernes

Ce site regroupe les notes de cours, les laboratoires, les devoirs, les ressources et les PDF du cours Création de sites web. Il peut être publié directement sur GitHub Pages.

Voir les semaines Ouvrir le plan du cours

Plan du cours

Tu peux remplacer ce texte par la description officielle de ton cours, ses objectifs, ses compétences et ses modalités d’évaluation.

Objectifs

  • Comprendre la structure d’une page web.
  • Créer des interfaces avec HTML et CSS.
  • Ajouter de l’interactivité avec JavaScript.

Évaluation

  • Laboratoires pratiques
  • Mini-projets
  • Travail final

Technologies

  • HTML5
  • CSS3
  • JavaScript
