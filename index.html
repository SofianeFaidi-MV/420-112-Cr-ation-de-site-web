Introduction à la programmation Web

Structure de base HTML

HTML sert à créer la structure d’une page Web. 

<!DOCTYPE html>
<html>
<head>
<title>Ma page</title>
</head>

<body>

<h1>Bonjour</h1>
<p>Bienvenue sur mon site</p>

</body>
</html>

Introduction au CSS

CSS permet de styliser une page Web en ajoutant couleurs, polices et mise en page. 

body {
background:black;
color:white;
font-family:Arial;
}

Introduction à JavaScript

JavaScript est un langage de programmation permettant de rendre une page Web interactive et de manipuler les éléments HTML grâce au DOM. :contentReference[oaicite:1]{index=1} 

alert("Bonjour !");

Variables JavaScript

Les variables servent à stocker des données.

let note = 85;

if(note >= 60){
console.log("Réussi");
}else{

}

Boucles

for(let i=0;i<5;i++){
console.log("Bonjour");
}
