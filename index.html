Création de sites Web

Introduction

Dans ce cours nous allons apprendre à créer des sites web modernes en utilisant :

  • HTML
  • CSS
  • JavaScript

HTML

HTML permet de structurer une page web. 

<h1>Bonjour</h1>
<p>Bienvenue sur mon site</p>

CSS

CSS permet de styliser une page web. 

body {
  background: black;
  color: white;
}

JavaScript

JavaScript permet d'ajouter de l'interactivité. 

function direBonjour(){
  alert("Bonjour !");
}
